## Add NuxtHub to Your Project

::code-group

```bash [pnpm]
pnpm add @nuxthub/core@latest
pnpm add -D wrangler
```

```bash [yarn]
yarn add @nuxthub/core@latest
yarn add -D wrangler
```

```bash [npm]
npm install @nuxthub/core@latest
npm install -D wrangler
```

```bash [bun]
bun add @nuxthub/core@latest
bun add --dev wrangler
```

::

## Add NuxtHub to Modules

```ts
// nuxt.config.ts
export default defineNuxtConfig({
  modules: ["@nuxthub/core"],
  hub: {
    db: 'sqlite',
  },
});
```

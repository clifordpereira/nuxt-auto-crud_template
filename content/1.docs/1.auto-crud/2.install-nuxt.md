---
title: Install Nuxt.js
description: For detailed steps, refer to https://nuxt.com/docs/getting-started/installation.
---

Run the following command to create a new Nuxt 3 project using the default UI template:

```bash
npx nuxi init -t ui <project_name>
```

Use the Nuxt Image module for efficient image optimization and manipulation:

::code-group

```bash [pnpm]
pnpm add @nuxt/image
```

```bash [yarn]
yarn add @nuxt/image
```

```bash [npm]
npm install @nuxt/image
```

```bash [bun]
bun add @nuxt/image
```

::

If you require a lightweight CMS, install the Nuxt Content module:

::code-group

```bash [pnpm]
pnpm add @nuxt/content
```

```bash [yarn]
yarn add @nuxt/content
```

```bash [npm]
npm install @nuxt/content
```

```bash [bun]
bun add @nuxt/content
```

::
